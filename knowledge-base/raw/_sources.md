# Source Registry

> Every raw source ingested into the knowledge base is registered here.
> The LLM uses this file to track what has and hasn't been compiled into the wiki.

---

## Format

```
### [Source ID] — Title
- **File:** `raw/filename.md`
- **Type:** article | paper | transcript | dataset | notes | image | repo
- **Date ingested:** YYYY-MM-DD
- **Description:** Brief description of the source
- **Status:** unprocessed | compiled | needs-update
- **Wiki links:** [[links to wiki articles generated from this source]]
```
